# patter

> Pipe stdout directly to [Mattermost](https://mattermost.com/).

## Install
*coming soon*
```
pip install patter
```

## Usage

> Send a message to a user.
```
$ echo "testing" | patter -u some_user
```

> Send a message to a channel.
```
$ echo "testing" | patter -c town-square
```
